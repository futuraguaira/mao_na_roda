# Diretiva da instância

## Propósito
Manter uma base operacional simples para o projeto Mão na Roda, com instruções claras e execução determinística.

## Fluxo recomendado
1. Revisar a intenção do pedido.
2. Consultar esta diretiva antes de editar arquivos.
3. Executar o script apropriado em execution/ quando houver automação.
4. Validar a mudança com um comando de verificação.

## Saídas esperadas
- Código funcional e consistente com o produto.
- Documentação mínima para manter a instância operante.
- Arquivos intermediários em .tmp/.
